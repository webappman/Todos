# RESTful JSON API and Automatic Test using Rspec

Build a RESTful JSON API With Rails 5 and Automatic Test using Rspec.

## Install

* Install Ruby -v 2.4.1 and Rails -v 5.0.1

* bundle install

* rake db:create db:migrate

* bundle exec rspec
